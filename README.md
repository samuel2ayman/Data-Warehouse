## 📊 SQL Data Warehouse (Medallion Architecture)

Built a SQL Data Warehouse using the Medallion Architecture (Bronze, Silver, Gold) to process data from CRM and ERP CSV sources.

- **Bronze:** Raw data ingestion from CSV files  
- **Silver:** Data cleaning, transformation, and validation  
- **Gold:** Aggregated, business-ready data for analytics  

The Gold layer is used for:
- 📈 Reporting & dashboards  
- 🤖 Machine learning  
- 🔍 Ad-hoc analysis  

**Tech:** SQL, Data Warehousing concepts
